# Deep Learning Course (code number: 40-715) – Sharif University of Technology

This repository is dedicated to the **Deep Learning** course offered by  
**Dr. Mahdieh Soleymani** at **Sharif University of Technology**,  
Master’s Program in **Artificial Intelligence**,  
**Faculty of Computer Engineering**.

## 📘 About This Repository

In this repository, we systematically cover the materials taught throughout the course.  
For each section of the course, you will find:

- 🧠 **Core deep learning principles and models**
- 🧪 **Practical assignments and hands-on exercises**
- ✍️ **Theoretical and analytical problem sets**

The goal of this repository is to provide a **well-structured and comprehensive reference** for students and learners who want to deeply understand both the **theoretical foundations** and **practical implementations** of deep learning.

## 🎯 Target Audience

This repository is suitable for:

- Graduate students in **Artificial Intelligence / Computer Engineering**
- Researchers and practitioners interested in **Deep Learning**
- Learners seeking a structured academic approach to DL concepts

## 🧩 Structure

Each section of the repository corresponds to a specific topic covered in the course and includes the relevant materials and exercises.
